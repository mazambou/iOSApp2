# Scavenger Hunt App

## Overview

Scavenger Hunt App is an iOS application developed using SwiftUI as part of Assignment 3 for the MWD3A (iOS Development) course.

The application allows users to participate in a city-wide scavenger hunt sponsored by local businesses. Users receive clues, locate hidden items, take photos as proof, and earn rewards based on the number of items found.

---

## Features

- Display a list of 10 hidden items
- View clues for each item
- Mark items as found
- Capture photos of found items
- Track hunt progress
- Calculate rewards based on completed items
- Submit results when the hunt is completed
- Modern SwiftUI interface
- Uses assets and images
- Fully commented source code

---

## Reward System

| Items Found | Reward |
|------------|----------|
| 5 or more | 10% Discount Code |
| 7 or more | 20% Discount Code |
| All 10 Items | 20% Discount Code + Entry into $5000 Grand Prize Draw |
